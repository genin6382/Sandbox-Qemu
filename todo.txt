Backend:
Endpoints:
• POST /nodes → create overlay
• POST /nodes/:id/run → start VM
• POST /nodes/:id/stop → stop VM
• POST /nodes/:id/wipe → wipe overlay
• GET /nodes → list nodes + status + Guacamole URL
• Overlay creation:
  qemu-img create -f qcow2 -b /images/base.qcow2 /overlays/node_<id>.qcow2
• QEMU runs with VNC enabled
• Guacamole auto‑register for node consoles

Guacamole:
• Run via Docker Compose
• Configured for VNC → QEMU VMs
• URL format: http://localhost:8080/guacamole/#/client/<connection-id>
